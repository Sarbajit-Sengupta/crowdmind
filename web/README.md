# CrowdMind

CrowdMind is an AI crowd intelligence platform for FIFA World Cup 2026 operations teams.

It analyzes attendance, weather, rivalry intensity, transit load, and historical incidents to predict crowd risk, identify hotspots, generate recommendations, and produce operational situation reports.

## Features

- Match risk scoring
- Dynamic match selector
- Crowd hotspot detection
- Stadium risk map
- Historical incident memory
- Situation report generator
- Operations Copilot
- Scenario simulator

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- MongoDB Atlas
- Google Gemini planned for AI recommendations
